Notes on attribute shape:
- `LibraryImport` is the modern source-generated alternative to `DllImport`. In .NET 8+ it supports `StringMarshalling = Utf8` natively.

- The `EntryPoint` syntax uses the WIT shape: `interface-path` as Library, kebab-case function name as EntryPoint.

## Task 4: Create `MobileBridgeHost.kt` (handlers + Defaults + Registration) **[AGENT]**

**Files:** Create `src/BlazorNative.Jni/src/main/kotlin/io/blazornative/jni/MobileBridgeHost.kt`.

**Step 1: Write the file.**

```kotlin
package io.blazornative.jni

import com.sun.jna.Pointer

/**
 * Phase 2.3 mobile_bridge host handler infrastructure.
 *
 * MobileBridgeHandlers holds per-import Kotlin implementations. Each
 * WasiHost caller (JVM tests, Android MainActivity) provides their own
 * handlers — Defaults.handlers is the JVM-stub fallback for tests that
 * don't need real data.
 *
 * Phase 2.3 has just shell-platform-info; future imports add fields per
 * docs/BACKLOG.md's deferred list (Phase 2.5 navigate/storage, Phase 4+
 * fetch).
 */
data class MobileBridgeHandlers(
    val platformInfo: () -> String
)

/**
 * Stub defaults — used when no MobileBridgeHandlers is explicitly passed
 * to WasiHost.loadAndRun. Returns a fixed JVM-stub JSON so BootSmokeTest
 * (JVM unit test) can assert on a known marker shape without needing to
 * pass a real implementation.
 */
object Defaults {
    val handlers = MobileBridgeHandlers(
        platformInfo = { """{"os":"jvm-default","note":"stub-host"}""" }
    )
}

/**
 * Registers all MobileBridgeHandlers against the component linker as
 * WIT-typed functions under the "blazornative:mobile-bridge/bridge"
 * interface name. Holds strong references to JNA Callbacks for the
 * linker's lifetime (prevents GC freeing the native trampolines while
 * wasmtime still has the pointer).
 */
class MobileBridgeRegistration(private val handlers: MobileBridgeHandlers) {
    private val callbackHolds = mutableListOf<WasmtimeComponentFuncCallback>()

    fun register(linker: Pointer) {
        val b = WasmtimeBindings.INSTANCE
        val ifaceName = "blazornative:mobile-bridge/bridge"

        val cbPlatformInfo = WasmtimeComponentFuncCallback { _, _, _, _, results, _ ->
            try {
                val info = handlers.platformInfo()
                WasmtimeBindings.writeStringResult(results, 0, info)
                null  // null = success (no trap)
            } catch (t: Throwable) {
                System.err.println("[mobile_bridge platformInfo handler threw] ${t.javaClass.simpleName}: ${t.message}")
                // Returning null lets wasmtime propagate as if the call succeeded
                // with whatever (likely garbage) is in results — for Phase 2.3 we
                // don't expect the handler to throw because it's a pure JSON literal.
                null
            }
        }
        callbackHolds += cbPlatformInfo  // keep alive

        val err = b.wasmtime_component_linker_define_func(
            linker,
            ifaceName, ifaceName.length.toLong(),
            "shell-platform-info", "shell-platform-info".length.toLong(),
            cbPlatformInfo, null, null
        )
        if (err != null) throw WasmtimeException.fromErrorPointer("define_func 'shell-platform-info'", err)
    }
}
```

/**
 * JNA callback wrapper for wasmtime_component_func_callback_t. Signature
 * (approximate; confirm per val.h):
 *
 *   typedef wasm_trap_t* (*wasmtime_component_func_callback_t)(
 *       void* env,

 *       const wasmtime_component_val_t* args, size_t nargs,
 *       wasmtime_component_val_t* results, size_t nresults);

        args: Pointer,

**Step 4: Add the `writeStringResult` helper** as a companion-object function on `WasmtimeBindings`:

```kotlin
    companion object {
        val INSTANCE: WasmtimeBindings = Native.load("wasmtime", WasmtimeBindings::class.java)

        /**
         * Writes a Kotlin String into a `wasmtime_component_val_t*` result
         * slot. Uses wasmtime_component_val_string_new which:
         *   1. Sets the val's discriminant to "string"
         *   2. Allocates a new UTF-8 buffer
         *   3. Copies the bytes into it
         *
         * The slot offset assumes wasmtime_component_val_t is a fixed-size
         * tagged union; the exact size TBD per val.h. For Phase 2.3 we use
         * a conservative 128-byte stride.
         */
        private const val COMPONENT_VAL_T_SIZE = 128L

        fun writeStringResult(results: Pointer, slot: Int, value: String) {
            val bytes = value.toByteArray(Charsets.UTF_8)
            val slotPtr = results.share(slot * COMPONENT_VAL_T_SIZE)
            INSTANCE.wasmtime_component_val_string_new(slotPtr, bytes, bytes.size.toLong())
        }
    }
```

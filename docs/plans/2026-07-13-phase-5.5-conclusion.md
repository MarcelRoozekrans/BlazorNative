## 2. The three doc-vs-CI reconciles (honesty fixes this gate)

The audit swept the current-state prose against what CI actually gates and found three
lags, all in docs behind a correct shipped workflow:

1. **DoD #4 "two-job" → "single job".** The DoD headline estimated a two-job workflow
   (mirroring Android's Windows-publish → Ubuntu-emulator split); the shipped `ios.yml`
   is a **single macOS job** because iOS publishes AND tests on the same runner. The
   substance (an informational macOS lane with the 9-export + 4-IL2072 + XCTest
   assertions and promotion criteria) is fully delivered. **Corrected in MILESTONE.md
   DoD #4** (authorized one-line honesty fix).
2. **iOS XCTest count "12" → "13".** `ios.yml` asserts **13 passed** (9 at 5.3 + 3
   clipboard at 5.4 + the `bn_bridge_callbacks` struct-drift pin). The "12" prose lagged
   because the 13th test (commit `c9ac4f2`) landed in PR #51 *after* the 5.4 conclusion
   was written. CI is the ground truth. **Corrected the MILESTONE.md DoD #6 count
   string**; the historical conclusion/ROADMAP prose stays as period record.
3. **.NET "2 skipped" → "0 skipped".** `ci.yml` asserts 230 passed / **0 skipped**
   (zero skips since M4 close). The audit's counts use 0; no doc edit needed (the raw
   counts in MILESTONE/ROADMAP were already correct).

These are exactly the wording-level residual risks the design anticipated — every DoD
was closed incrementally with per-phase evidence, so nothing substantive surfaced.

## 4. The M5 arc (what shipped)

Six phases turned a single-shell architecture claim into a two-platform reality:

- **5.0** proved the named risk GREEN (and boots on the sim) — the runtime-pack bypass
  ports from linux-bionic to iOS.
- **5.1** gave the Android shell real host-initiated ingress (the 9th export;
  lifecycle + predictive-back-→-NavigateBack + launch deep link), closing the 4.2
  `NativeEvents` fork.
- **5.2** built the first non-Kotlin shell (Swift/UIKit) and rendered BnDemo on the CI
  simulator — the static-`.a`-into-Xcode link recipe was the hard-won discovery.
- **5.3** made it interactive — iOS reached the Android v3.0 bar with **zero shared
  change**.
- **5.4** landed clipboard + share on both platforms through a documented,
  size-negotiated bridge that grows without an ABI break.
- **5.5** (this phase) audited it all and closed the milestone.

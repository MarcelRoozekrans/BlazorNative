# Phase 3.0e — WASM-Era Collapse — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Delete the WASM/wasmtime runtime era, rework the golden tests to typed expectations (unlocking full kotlinx-serialization removal), and rename `BlazorNative.NativeHost` → `BlazorNative.Runtime` end-to-end including the JNA library name.

**Architecture:** Three staged commits-groups on one branch: Stage 1 delete (Tasks 1–6, gated by all three test surfaces), Stage 2 rename (Tasks 7–9, gated by fresh publishes + all surfaces under the new name), Stage 3 docs (Task 10). Complete reference inventory: [2026-07-09-phase-3.0e-design.md](2026-07-09-phase-3.0e-design.md) — every deletion below is backed by the sweep in that doc.

**Tech Stack:** Same pinned toolchain as 3.0c/d (SDK 10.0.3xx, NativeAOT runtime packs 10.0.9, NDK 26.3, JNA 5.14, AGP 8.7.3, AVD `blazornative-pixel6-x86_64`).
